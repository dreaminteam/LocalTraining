Правила


Основные законы, действующие в организации конных забегов. Участники, наездники, тренеры, должности, правила дисквалификации.

  1. Перед забегом каждая лошадь должна показать себя на смотре перед стартом.
  2. Участница может стартовать после взмаха флажком, из киоска или от ворот.
  3. Допускается рестарт, если был зафиксирован фальстарт.
  4. Победителем в гонке считает именно тот участник, что первым преодолел финишную прямую.
  5. В случае, когда финишную прямую одновременно пересекли несколько участников, победитель определяется по фотофинишу.
  6. Победителями могут быть два участника одновременно при равных условиях пересечения финишной прямой, в таком случае призовые средства делят пополам и присуждают каждой из лошадей.

Должностные лица

Должностное лицо, контролирующее забеги, имеет название — стюард.

Другие лица:

  1. Секретарь. Он несет ответственность за трек, занимается планированием и делает «махинации» для забега.
  2. Гандикапер. Готовит дополнительный вес для скачек по правилам гандикапа.
  3. Весовщик. Ответственен за процесс взвешивания участников скачек.
  4. Судья. Оглашает окончательный результат забега.
  5. Начинающий. «Стартует» скачки.

Правила для лошадей

  1. Возраст коня принято отсчитывать с первого января того года, когда он появился на свет, а не с момента его рождения.
  2. На плоском треке может работать лошадь от двух лет и не раньше.
  3. На треках с низкими барьерами может работать лошадь не ранее трех лет.
  4. На треках с высокими барьерами может работать лошадь не ранее четырех лет.
  5. Не разрешается применять капюшоны для того чтобы ослеплять скакунов.
  6. Имя лошади для скачек не должно быть более восемнадцати букв.

Законы по дисквалификации на лошадиных бегах

  1. Если участница забега сталкивается в ходе скачек с другой участницей, ее имеют право дисквалифицировать.
  2. Участница забега может быть дисквалифицирована также в случае неправомерных действий своего наездника.
  3. Дисквалификации подвергается та лошадь, чей наездник оказался в весе после скачек меньшем, чем после взвешивания перед стартом.
  4. Дисквалификации подлежит участник, который прекращает слушать команды перед началом скачек.

Жокейские положения во время скачек

  1. Без лицензии наездник к скачкам не допускается
  2. Взвешивание жокея происходит до старта на скачках.
  3. Шлем для защиты головы обязателен для скачек для наездника.
  4. Обмундирования наездника должно соответствовать символике владельца лошади.
  5. Не имеет значения, до или после процедуры взвешивания наездника имеют право заменить другим.
  6. Главное правило наездника – сделать все возможное и невозможное, для того чтобы получить хороший результат от лошади на скачках.

Тренер и его должностные обязанности:

  1. Главная часть ответственности тренера – общий вес участника и контроль за ним. Сюда включается и вес наездника, и сбруи, и прочих атрибутов.
  2. Также тренер контролирует состояние седла, в случае если наездник – в статусе ученика.
